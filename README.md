# Economic Report Generator

경제 뉴스 텍스트를 분석하여 구조화된 리포트로 변환해주는 웹 애플리케이션입니다.

## 주요 기능

- 구조화된 경제 뉴스 텍스트 파싱
- 주차별 리포트 생성
- 중요 항목 강조 표시
- 공유 URL 생성
- PNG 이미지로 리포트 다운로드

## 사용 방법

1. 텍스트 입력 창에 경제 뉴스 텍스트 붙여넣기
2. "Generate Report" 버튼 클릭
3. 생성된 리포트 확인 및 공유

## 텍스트 형식

경제 뉴스 텍스트는 다음과 같은 형식을 따라야 합니다:

```
♧ 2.3 月 News & Trends ♧
"주요 인용구"

Ⅰ. 첫 번째 섹션 제목
1. 하위 섹션 제목
ㆍ뉴스 항목 1
ㆍ뉴스 항목 2
📍 중요한 뉴스 항목

2. 두 번째 하위 섹션
ㆍ뉴스 항목 1
ㆍ뉴스 항목 2

Ⅱ. 두 번째 섹션 제목
...
```

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 빌드
npm run build
```

## 기술 스택

- React
- TailwindCSS (CDN)
